# /collect 스킬

## 사용법
```
/collect <지역> <키워드> [수량]
```

## 예시
```
/collect 서울 제조공장 50
/collect 부산 병원 30
```

## 실행 방법
사용자가 제공한 인자를 파싱하여 아래 스크립트를 실행하세요:

```bash
npx tsx scripts/collect.ts <지역> <키워드> [수량]
```

## 실행 후
- 스크립트의 JSON 출력을 읽고 사용자에게 결과를 보고하세요
- 수집된 리드 수, 지역, 키워드를 명확히 알려주세요
- 에러가 발생하면 에러 메시지를 전달하세요
