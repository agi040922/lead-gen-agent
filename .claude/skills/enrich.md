# /enrich 스킬

## 사용법
```
/enrich [lead-id]
```

## 예시
```
/enrich                    # 웹사이트 있고 아직 분석 안 한 리드 전체 보강
/enrich abc-123-def        # 특정 리드만 보강
```

## 실행 방법
```bash
npx tsx scripts/enrich.ts [lead-id]
```

## 동작 방식
1. 리드의 웹사이트 메인 페이지 크롤링
2. 내부 링크 추출 → 우선순위 정렬 (채용/문의/회사소개 페이지 우선)
3. 최대 15개 페이지에서 이메일 패턴 추출
4. 웹사이트 기술 분석 (HTTPS, 반응형, 서버, 기술스택)
5. DB 업데이트 (이메일, 분석 결과)

## 실행 후
- 발견된 이메일과 크롤링 페이지 수를 보고하세요
- 웹사이트 기술 분석 결과도 간단히 알려주세요
