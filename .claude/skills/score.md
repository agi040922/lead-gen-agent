# /score 스킬

## 사용법
```
/score [lead-id]
```

## 예시
```
/score                    # 점수 0인 모든 리드 점수 매기기
/score abc-123-def        # 특정 리드만 점수 매기기
```

## 실행 방법
```bash
npx tsx scripts/score.ts [lead-id]
```

## 실행 후
- 점수를 매긴 리드 수를 보고하세요
- 점수 기준을 간단히 설명해주세요 (웹사이트 없음 +30, 구형 웹사이트 +20, 운영 불편 리뷰 +20, 적정 규모 +15, 이메일 있음 +15)
